Markup

This is the base markup but some attributes and elements can be removed depending on how the field is going to behave. These are some rules to follow:

<div class="sui-form-field">

	<label for="unique-id" id="unique-id-label" class="sui-label">Field label</label>

	<select
		id="unique-id"
		class="sui-search"
		multiple
		data-placeholder="Placeholder text"
		aria-required="true"
		aria-labelledby="unique-id-label"
	>
		<option value="option-1">Option 1</option>
		<option value="option-2">Option 2</option>
		<option value="option-3">Option 3</option>
		<option value="option-4">Option 4</option>
		<option value="option-5">Option 5</option>
	</select>

</div>

Include sui-form-field-error class next to sui-form-field in order to get error state styles, also add content to unique-id-error-message element and show it. Focus back field, or the beginning of the form if part of some, for screenreader to read again the fields with error message.

Re-init

If you added this element after the DOM has loaded you can initialize it again by calling:

if ( $( '.sui-color-accessible' )[0] ) {

	$( '.sui-search' ).SUIselect2({
		minimumInputLength: 2,
		maximumSelectionLength: 1,
		dropdownCssClass: 'sui-search-dropdown sui-color-accessible'
		placeholder: function() {
			$( this ).data( 'placeholder' );
		},
	});
} else {

	$( '.sui-search' ).SUIselect2({
		minimumInputLength: 2,
		maximumSelectionLength: 1,
		dropdownCssClass: 'sui-search-dropdown'
		placeholder: function() {
			$( this ).data( 'placeholder' );
		},
	});
}
