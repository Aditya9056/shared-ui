The Shared UI library supports styling of the jQuery UI Datepicker but doesn't include the library so you will need to include that in your plugin yourself. To use the styles you will need to add the class sui-calendar to your datepicker element.

See code snippet below the sample to see how it was done in this showcase.

$( '.sui-date input' ).datepicker({
	beforeShow: function( input, inst ) {
		$( '#ui-datepicker-div' ).addClass( 'sui-calendar' );
	},
	'dateFormat': 'd MM yy'
});
