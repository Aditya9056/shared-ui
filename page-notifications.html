Documentation

Alert Notices are ARIA live regions that are known for not working on page load, but instead need the region to be updated after page has loaded.

Base Markup

<div role="alert" class="sui-notice" aria-live="assertive">
	...
</div>

Floating Notices

For a notice to be floated it needs to be placed inside sui-floating-notices wrapper. For even better accessibility, add this wrapper after sui-header element closing tag.

<div class="sui-floating-notices">
	...
</div>

HTML Actions

Open Notice

Place the following attributes on an element to trigger open notice function:

  1. This attribute will open the alert notice you assign:
    data-notice-open="unique-id" required
  2. Use this attribute to type the message that's going to be displayed in the alert notice:
    data-notice-message="Type message here" required
     
    NOTE: You can also print HTML content inside as part of data-notice-message value, for example you can make some text bold:
    "Message with <strong>bold text</strong>"
    Or you can include a second paragraph by closing the one that's added by SUI and open a new one, like this:
    "First paragraph</p><p>Second paragraph"
  3. Use this attribute if you need to replace default "info" icon, but have in mind to not use full class name only what's after "sui-icon-":
    data-notice-icon="info"
    NOTE: Spin animation for "loader" icon will be applied automatically.
  4. The following attribute will work differently depending on the type of alert notice you want to show: floating or inline.
    • Floating Notices:
      These notices autoclose by default after 3000 milliseconds (3 seconds) but if you need to modify the timing use the following attribute:
      data-notice-autoclose="5000"
      If you need to turn off autoclose ability, simply use "off" or "false" text as this attribute value.
    • Inline Notices:
      These notices don't have autoclose ability but if you need to enable this simply include the attribute and make sure the value entered is in milliseconds:
      data-notice-autoclose="5000"
  5. Some notices need a dismiss button, when this happen you can add the attribute below with "true" value:
    data-notice-dismiss="true"
  6. This attribute is linked to the previous one, when you have dismiss button enabled you can add a tooltip to it:
    data-notice-dismiss-tooltip="Dismiss" 
<button
	data-notice-open="unique-id"
	data-notice-message="Type message here"
	data-notice-icon="info"
	data-notice-autoclose="off"
	data-notice-dismiss="true"
	data-notice-dismiss-tooltip="Dismiss"
>
	Open notice
</button>
Close Notice

To dismiss (close) an alert notice add data-notice-close="" attribute to an element inside the notice that's going to trigger the closing action.

<button data-notice-close="">Close this notice</button>

JS Actions

Open Notice

Trigger the function below when you need to notify a message to users:

SUI.openNotice( noticeId, noticeMessage, noticeIcon, noticeAutoClose );
  1. @param noticeId required
    The unique ID of the element serving as the alert notice container.
  2. @param noticeMessage required
    You only need to include here the text that's going to be displayed in the notice.
     
    NOTE: You can also print HTML content inside as part of the paramether value, for example you can make some text bold:
    let noticeMessage = 'Message with <strong>bold text</strong>';
    Or you can include a second paragraph by closing the one that's added by SUI and open a new one, like this:
    let noticeMessage = 'First paragraph</p><p>Second paragraph</p>';
  3. @param noticeIcon
    Use this paramether to replace default "info" icon on your alerts. You can set it to `null` or keep it empty if you don't want or need to use this variable:
    NOTE: Spin animation for "loader" icon will be applied automatically.
  4. @param noticeAutoClose
    This paramether works differently depending if the alert notice floats or goes inline with the content.
    • Floating Notice:
      This kind of notice autoclose after 3000ms (3 seconds) by default but you can modify this by setting another value in milliseconds.
      If you need to turn off autoclose ability, use "off" or "false" value for the paramether.
    • Inline Notice:
      This kind of notice never autoclose, but if you want it to have this ability simply include a value in milliseconds. You can set it to `null` or keep it empty if you don't need to use this variable.
  5. @param noticeDismiss
    This paramether when its value is set to true will add a dismiss button to the notice.
  6. @param noticeDismissTooltip
    This paramether works along with previous one. When dismiss button is enabled, you can add a tooltip that will appear when user hovers the dismiss button. Fill this paramether value with the text you want to display. 
( function( $ ) {

	const button = $( 'button' );

	button.on( 'click', function() {

		let self                 = $( this ),
			noticeId             = 'unique-id', // Do not include number sign.
			noticeMessage        = 'Type message here',
			noticeIcon           = 'loader',
			noticeAutoClose      = 5000,
			noticeDismiss        = true,
			noticeDismissTooltip = 'Dismiss'
			;

		SUI.openNotice( noticeId, noticeMessage, noticeIcon, noticeAutoClose, noticeDismiss, noticeDismissTooltip );

	});
}( jQuery ) );
Close Notice

To dismiss (close) an alert notice use the function below triggered by an element that must be inside notice element.

SUI.closeNotice( this );
