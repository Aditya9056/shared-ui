Float Notifications

The top notifications have different styling than inline notifications and are fixed positioned in the screen. They use the class sui-notice-top or sui-notice-floating in the parent class instead of sui-notice.

Click on the buttons below to try both notifications:

By default sui-notice-top notification will automatically hide after 3 seconds. If you prefer to let users hide the notification by themselves you can add sui-can-dismiss class to the parent div and add a dismiss button to the notice. If you don't want the notice dismissed at all you can add sui-cant-dismiss class to the parent div.

Below are code samples of both: sui-notice-top and sui-notice-floating (floating with button):

<!-- Top notice with button. -->
<div class="sui-notice-top sui-notice-success sui-can-dismiss">
	<p><strong>Notice success:</strong> Page caching is currently active.</p>
	<span class="sui-notice-dismiss">
		<a role="button" href="#" aria-label="Dismiss" class="sui-icon-check"></a>
	</span>
</div>

<!-- Floating notice with button. -->
<div class="sui-notice-floating sui-notice-info">
	<p>You've made changes to your file structure</p>
	<div class="sui-notice-buttons">
		<button href="#" class="sui-button sui-button-blue"><i class="sui-icon-web-globe-world" aria-hidden="true"></i>Publish</button>
	</div>
</div>
